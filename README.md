# SHOOTING GAME
---

## 1. SYSTEM OVERVIEW

The game features a wave-based combat system where the player fights increasingly stronger monsters. After several waves, a boss appears; defeating it completes the level. The player can then choose to continue with harder waves or return to the lobby.
Monsters drop gold and experience when defeated. Gaining experience levels up the character, increasing their attack power to face tougher challenges.
